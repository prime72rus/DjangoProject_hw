## Использование
Загрузите репозиторий по ссылке:  
[https://github.com/prime72rus/DjangoProject_hw](https://github.com/prime72rus/DjangoProject_hw)

Для запуска и работы приложения установите зависимости указанные в файле *pyproject.toml*


### Запуск Development сервера
Чтобы запустить сервер для разработки, выполните команду:
```
python manage.py runserver 8000
```
